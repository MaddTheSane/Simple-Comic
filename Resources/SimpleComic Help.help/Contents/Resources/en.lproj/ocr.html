Selectable Text

On Apple Silicon Macs Simple Comic uses Apple's Optical Character Recognition, to make text in comics selectable for copying and text-to-speech. Apple calls this feature “Live Text.”

  • The mouse cursor changes from the arrow cursor to the I-beam cursor when over selectable text.
  • Click-and-drag to select live text. Recognized words within the dragged out rectangle are selected.
  • Copy, Select All, and Speak, on the Edit menu work on selectable text.
  • Control-click on selected text for a contextual menu with Copy, Look Up, and Speak.
  • ⌘-click-and-drag to add a new selection rectangle to the existing selection.
